# Student Performance Predictor #

📌 Overview
This project predicts student marks based on study hours, sleep hours, and attendance using Machine Learning.

🚀 Features
- Simple and easy to use
- Uses Linear Regression
- Takes user input
- Predicts marks instantly

🛠️ Technologies Used
- Python
- Pandas
- Scikit-learn

 ▶️ How to Run
1. Install dependencies:
pip install pandas scikit-learn
2. Run the program:
python app.py
3. Enter values when prompted

📊 Example
Input:
Study hours: 5  
Sleep hours: 7  
Attendance: 80  

Output: Predicted Marks: 65.00
